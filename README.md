# PlandIt

A calendar application for uOttawa students.

## Features

- Calendar views (Month/Week)
- Event management
- Light/Dark themes
- Responsive design
- Password protection
- uOttawa themed

## Files

- index.html - Home page
- calendar.html - Calendar
- login.html - Login
- register.html - Registration
- team.html - Team info
- ProductFeatures.html - Features
- style.css - Styles
- home-calendar.js - Calendar code
